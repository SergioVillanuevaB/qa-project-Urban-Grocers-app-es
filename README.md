# Proyecto Urban Grocers 


Descripción
- Proyecto de testing de API enfocado en el backend de Urban Grocers, validando funcionalidades clave como gestión de kits y servicios de entrega. Se verificó el correcto funcionamiento de los endpoints, la integridad de los datos y el cumplimiento de la lógica de negocio.

Técnicas de testing
- Pruebas positivas y negativas
- Partición de clases de equivalencia
- Análisis de valores límite
- Validación de API (requests/responses)
- Validación de lógica de negocio

Tecnologías y herramientas
- Postman
- JSON
- Métodos HTTP (GET, POST, PUT, DELETE)
- Jira
- ApiDoc
- Google Sheets / Docs
  
Como usarlo
- Necesitas tener instalados los paquetes pytest y request para ejecutar las pruebas.
- Ejecuta todas las pruebas con el comando pytest.
